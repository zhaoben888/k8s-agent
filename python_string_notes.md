# 🐍 Python 字符串操作精华笔记

### 1. 格式化与基础 (Formatting & Basics)
| 语法 | 功能 | 示例 | 结果 |
| :--- | :--- | :--- | :--- |
| **`f"{var}"`** | **f-string**：嵌入变量 | `name="Tom"; f"Hi {name}"` | `"Hi Tom"` |
| **`len()`** | **长度**：字符或元素个数 | `len("ABC")` | `3` |

### 2. 位置与截取 (Positioning)
| 语法 | 功能 | 示例 | 结果 |
| :--- | :--- | :--- | :--- |
| **`s[i]`** | **索引**：取第 i 个字符 | `"Python"[0]` | `"P"` |
| **`s[a:b]`** | **切片**：取从 a 到 b-1 | `"Python"[0:2]` | `"Py"` |

### 3. 常用动作 (Common Methods)
| 方法 | 功能 | 记忆点 |
| :--- | :--- | :--- |
| **`.upper()`** | **变大写** | 动作：全部转大写字母 |
| **`.strip()`** | **去空格** | 动作：剥掉首尾“外皮”（空白） |
| **`.replace()`** | **替换** | 动作：旧换新 |
| **`.split()`** | **切割** | 动作：切碎成列表 (List) |
| **`.join()`** | **连接** | 动作：用连接符拼回字符串 |

### 4. 判断与逻辑 (Boolean Logic)
| 方法 / 关键字 | 场景 | 功能 | 结果 |
| :--- | :--- | :--- | :--- |
| **`.startswith()`** | `if s.startswith("a"):` | 检查开头 | `True/False` |
| **`in` (判断)** | `if "x" in s:` | 检查是否存在 | `True/False` |
| **`in` (循环)** | `for c in s:` | 逐个取出字符 | 依次操作每个字符 |

---

### 💡 核心规律总结：
*   **`.方法()`**: 都是在做一个**动作**或**提问**。
*   **`[下标]`**: 都是在**找位置**。
*   **`in`**: 既是**侦探**（判断），也是**搬运工**（循环）。
